# Continuance v1.1.1

This patch upgrades Continuance's default behavior so visual contemplation becomes part of the standard user experience rather than an optional flourish.

## What's new

### 1. Default image generation is now explicit
- Continuance now clearly defaults to generating a contemplative image for:
  - daily meditation
  - daily guidance
  - personal spiritual questions
  - existential or emotional struggles
- The skill instructions now treat image generation as core behavior, not optional decoration.

### 2. Images should match the user's inner atmosphere
- Guidance now emphasizes that images should express the user's **context, emotional terrain, and 境界**.
- Prompting guidance now prefers resonance over literal illustration.
- Added stronger direction for grief, burnout, hard choices, regret, and feeling stuck.

### 3. README now explains image API expectations
- Added setup guidance for OpenClaw image generation providers
- Clarified that the intended product experience is **image + guidance together**
- Added practical examples for daily meditation and personal life questions
- Documented typical applications and visual themes

## Why this matters

Continuance works best when the user receives not only text but also a contemplative visual field that carries the mood of the guidance. This patch makes that expectation explicit for both skill users and future maintainers.
